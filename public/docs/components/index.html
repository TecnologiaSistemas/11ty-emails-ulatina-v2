Ubicación: /src/_includes/components/basic_content.njk
Objeto de configuración: blockContent

Bloques a extender

  • beforeColumn

  • contentColumn

    • column

  • afterColumn

Parametros Object

  • table: Object ver macro table para las opciones

  • gutters: Boolean/Object ver macro gutter para las opciones

    • left: Object ver *macro gutter para las opciones, solo aplica para la columna izquierda y sobreescribe las opciones generales
    • right: Object ver macro gutter para las opciones, solo aplica para la columna derecha y sobreescribe las opciones generales

  • spacer: Boolean/Object ver macro spacer para las opciones

    • top: Object ver macro spacer para las opciones, solo aplica colocar un espacio en la parte superior y sobreescribe las opciones generales
    • bottom: Object ver macro spacer para las opciones, solo aplica colocar un espacio en la parte inferior y sobreescribe las opciones generales

  • responsive: Boolean true para colocar la etiqueta th y la clase column para hacer responsivo el bloque column, default ‘false’

Ubicación: /src/_includes/components/list.njk
Objeto de configuración: blockList
Extiende:: /src/_includes/components/basic_content.njk

Bloques a extender
  • topRow
  • bottomRow

Parametros Object

Usa todas las opciones del componente General Content más:

  • items: Object objeto con la colección de datos y configuración de la lista

    • type: String indica el tipo de lista a mostrar, valores aceptados md para archivos markdown, html para usar el tag <ul />, image para mostrar una imagen como índice, letter para mostrar letras como índice, number para mostrar números como índice, roman para mostrar números romanos como índice

    • spacer: Object ver macro spacer para las opciones, este se aplica solo cuando el html como resultado es una tabla. Se aplica en cada celda del renglón

    • content: Array arreglo de objetos con el texto a mostrar en la lista

      • file: String ruta del archivo dentro de src/_content cuando el tipo (type) es md
      • image: Object ver macro image para las opciones cuando el contenido sea imagen
      • text: String texto a mostrar, se omite si el tipo (type) es md

    • attributes: Object

      • class: String clase css a aplicar en cada elemento de la lista

      • index: Object valores para el índice cuando se usa el tag <table />, ver column para las opciones

        • align: String
        • class: String
        • valign: String
        • width: Number
        • prepend: String texto que se añade al principio del índice
        • append: String texto que se añade al final del

      • content: Object valores para el contenido cuando se usa el tag <table />, ver column para las opciones

        • align: String
        • class: String
        • valign: String
