setAttr

Ubicación: /src/_includes/components/functions.njk

Parametros

Se utliza normalmente para agregar, en caso de ser necesario, un atrinuto y su valor definido o asiganr un valor predeterminado. En caso de que ambos valores no esten definidos se omite.
  • attribute: String nombre del atrinuto a agregar
  • value: String/Number valor del atributo
  • default: String valor predeterminado del atriuto, opcional, default ‘’
Nunjucks

setAttrs ( attribute, value, default )

<tag {{ setAttrs(“width”, 500, “100%” ) }}“></tag>
<tag {{ setAttrs(“width”, “”, “100%” ) }}”></tag>
<tag {{ setAttrs(“width”) }}"></tag>
HTML

<tag width=“500”></tag>
<tag width=“100%”></tag>

setValue

Ubicación: /src/_includes/components/functions.njk

Parametros

Se utliza normalmente para especificar el valor de un elemento, si el valor dado es un objeto, es necesario especificar el índice para aplicarlo, si el valor es un número, texto o un valor boleano se aplica como esta espcificado, en caso de no existir un valor se omite.

  • value: String/Number valor del atributo

  • id: String nombre del índice en el objeto de datos
Nunjucks

setValue ( value, id )

<tag width=“{{ setValue({
  height: 100,
  width: 200
}, “width” ) }}”></tag>
<tag width=“{{ setValue(300) }}”></tag>
<tag width=“{{ setValue() }}”></tag>
HTML

<tag width=“200”></tag>
<tag width=“300”></tag>
<tag></tag>

column

Ubicación: /src/_includes/components/functions.njk

Parametros Object / Boolean

generic

Ubicación: /src/_includes/components/modules.njk

Parametros Object

  • class: String clase css para identificar a elemento de contenido utiliza setContent, default ‘’

  • table: Object ver macro table para las opciones

  • gutters: Boolean/Object ver macro gutter para las opciones generales

    • left: Object ver macro gutter para las opciones, solo aplica para la columna izquierda y sobreescribe las opciones generales

    • right: Object ver macro gutter para las opciones, solo aplica para la columna derecha y sobreescribe las opciones generales

  • spacer: Boolean/Object ver macro spacer para las opciones generales

    • top: Object ver macro spacer para las opciones, solo aplica colocar un espacio en la parte superior y sobreescribe las opciones generales

    • bottom: ver macro spacer para las opciones, solo aplica colocar un espacio en la parte inferior y sobreescribe las opciones generales

Caller
Nunjucks

{% call generic({
  class: “content”,
  table: true,
  gutters: { width: 40 },
  spacer: { size: 30 }
}) %}
  {{ caller() }}
{% endcall %}
HTML

<table width=“100%” cellspacing=“0” cellpadding=“0” border=“0”>
  <tbody>
    <tr>
      <td class=“content-spacing” width=“40” />
      <td style=“font-size: 0 !important; font-weight: normal; line-height: 0 !important; padding: 0 !important;”>
        <table class=“spacer” />
          <div class=“content”>
            {{ caller() }}
          </div>
        <table class=“spacer” />
      </td>
      <td class=“content-spacing” width=“40”/>
    </tr>
  </tbody>
</table>
