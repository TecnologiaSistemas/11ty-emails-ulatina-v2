Imagen y texto

Ubicación: /src/_includes/modules/header/image-text.njk
Objeto de configuración: blockHeader
Extiende: /src/_includes/components/header.njk

Parametros Object

Usa todas las opciones del componente General Content más:

  • image: Object ver macro image para las opciones

  • textPosition: String indica la posición del text a mostrar, valores aceptados “left” o “right”, default ‘left’

  • text: String texto a mostrar

  • responsive: Boolean true para colocar la etiqueta th y la clase column para hacer responsivo el header, default ‘false’

  • mobileSpacer: String clase css para indicar el espacio que solo se muestra en móviles cuando responsive = true, default ‘’

Imagen, texto e icono

Ubicación: /src/_includes/modules/header/image-icon-text.njk
Objeto de configuración: blockHeader
Extiende: /src/_includes/components/header.njk

Parametros Object

Usa todas las opciones del componente General Content más:

  • image: Object ver macro image para las opciones

  • textPosition: String indica la posición del text a mostrar, valores aceptados “left” o “right”, default ‘’

  • text: String texto a mostrar

  • icon: Object ver macro image para las opciones

    • gutter: Boolean/Object ver macro gutter para las opciones y componentes para implementación específicas

  • responsive: Boolean true para colocar la etiqueta th y la clase column para hacer responsivo el header, default ‘false’

  • mobileSpacer: String clase css para indicar el espacio que solo se muestra en móviles cuando responsive = true, default ‘’
