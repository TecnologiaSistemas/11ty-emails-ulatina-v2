Parametros Object

Usa todas las opciones del componente General Content más:

  • items: Array

    • table: Object ver módulo table para las opciones

    • image: Object ver macro image para las opciones

      • outerTable: Object agrega una tabla adicional como contenedor del elemento, ver nmacro generic para las opciones

    • cta: Object ver módulo CTA para las opciones

      • outerTable: Object agrega una tabla adicional como contenedor del elemento, ver macro generic para las opciones

    • text ó title: String texto a mostrar, el identificador title solo pretende mostrar de forma explicita cuando el texto colocado orresponde a un título

      • outerTable: Object agrega una tabla adicional como contenedor del elemento, ver macro generic para las opciones
