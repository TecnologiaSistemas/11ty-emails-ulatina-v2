Footer

Ubicación: /src/_includes/modules/footer.njk
Objeto de configuración: blockFooter
Extiende: /src/_includes/components/footer.njk

Parametros Object

Usa todas las opciones del componente General Content más:

  • text: String texto a mostrar antes de la información de contacto

  • image: Object objeto para el llamado y configuración de la imagen, ver *macro image para las opciones

  • contactInfo: String arreglo de objetos con la información de contacto, usa el macro imageContent. Cada elemento del arreglo genera una columna

    • align: String alienación de la tabla dentro de la columna
    • class: String clase css para identificar la tabla dentro de la columna
    • icon: Object objeto para el llamado y configuración de la imagen como icono, ver *macro image para las opciones
    • text: String texto que acompaña al icono

  • legal: String texto a mostrar al final del footer, normalmente corresponde al texto legal o copyright.

    • class: String clase css para identificar al elemento
    • text: String texto a mostrar

  • responsive: Boolean true para hacer responsivo
