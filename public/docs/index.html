Parametros Object

column / columnHead

Ubicación: /src/_includes/components/functions.njk

La única diferencia entre colum y columnHead es la etiqueta utilizada. column usa <TD> y columnHead usa <TH>

Parametros Object
  • align: String indica la alienación del conteido de la celda, default ‘’
  • bgcolor: String color de fondo de la celda, default ‘’
  • class: String clase css para identificar la celda y aplicar estilos en línea adicionales, default ‘’
  • colspan: Number número para establecer la cantidad de columnas a unir, default ‘’
  • height: Number número para establecer la altura de la celda, default ‘’
  • rowspan: Number número para establecer la cantida de renglones a unir, default ‘’
  • valign: String establece la alineación vertical de la celda, default ‘’
  • width: String o Number texto para indicar un porcentaje y número para anchos específicos, default ‘’

{% call function.column({
class: “demo”,
align: “center”,
valign: “top”,
bgcolor: “yellow”,
width: “100%”,
height: 40
}) %}
{{ caller() }}
{% endcall %}

<td align=“left” bgcolor=“red” class=“demo” width=“100%” height=“40” valign=“top” style=“font-size: 0 !important; font-weight: normal; line-height: 0 !important; padding: 0 !important;”>
{{ caller() }}
</td>

{% call function.columnHead() %}
{{ caller() }}
{% endcall %}

<th style=“font-size: 0 !important; font-weight: normal; line-height: 0 !important; padding: 0 !important;”>
{{ caller() }}
</th>
